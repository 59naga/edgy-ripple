# EdgyRipple

> Pixelated ripple effect using canvas

# Installation

```bash
$ bower install edgy-ripple
```

```html
<script src="bower_components/q/q.js"></script>
<script src="bower_components/edgy-ripple/edgy-ripple.js"></script>
<style>
  html,body,div{
    margin: 0;
    height: 100%;
  }
</style>
<div class="edgy-ripple">Kaboom!</div>
```
